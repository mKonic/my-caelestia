# my-caelestia

A personal Wayland desktop shell built on Hyprland and Quickshell. Started life as a fork of the [Caelestia](https://github.com/caelestia-dots) dots and shell, but has since diverged.


## Installation

```sh
git clone https://github.com/mKonic/my-caelesta
cd my-caelestia
./install.fish
```

## Luafication

As of Hyprland 0.55+, hyprlang is deprecated; this repo's Hyprland config is hand-ported to Lua.
